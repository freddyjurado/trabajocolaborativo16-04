Seguridad Informatica - UNAD ECBTI

Esta pagina web fue diseñada explicitamente para adquirir el conocimiento necesario sobre HTML5 y CSS por medio de ejercicios de practica... teniendo en cuenta esto, se tomo como objeto de estudio el Curso 204039 referente a Seguridad Informática, cuya tematica hace referencia a los Fundamentos seguridad informática.

Objetivos ds Seguridad informatica.
• Identificar amenazas de seguridad a las que se expone una red.
• Describir cada concepto de seguridad informática y definir el impacto que se tiene en la red al no tener un buen mecanismo de seguridad.
• Garantizar integridad, disponibilidad y privacidad de datos y recursos.

Objetivos OVA:
• Que el estudiante se fundamente teóricamente sobre las generalidades básicas del Diseño Web.
• Que el estudiante pueda utilizar de forma correcta los lenguajes HTML5 y CSS3 para aplicarlos en la solución de problemas de construcción de Sitios Web.
• Que el estudiante implemente servicios y sitios web a través de los sistemas gestores de contenidos CMS y LMS libres más usados en la actualidad.
